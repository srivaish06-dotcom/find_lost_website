# 🔍 Lost & Found Website

This is a simple and user-friendly Lost & Found web application that helps users report lost items and find items that have been found by others.

## 🚀 Features

- 📢 Report Lost Items  
- 📦 Add Found Items  
- 🔎 Search and browse items  
- 🧾 View item details  
- 📱 Responsive design for all devices  

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

## 💡 Project Description

The Lost & Found website is designed to connect people who have lost items with those who have found them. Users can easily submit details about lost or found items, making the recovery process faster and more efficient.

## 🎯 Objectives

- Simplify the process of reporting lost items  
- Help users recover their belongings  
- Provide an easy-to-use interface  
- Improve problem-solving and development skills
